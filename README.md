# Delivery-Cost-Analysis
## project overview 
This project aims to analyze the year 2024 delivery data to identify the key variables driving transportation costs and propose actionable insights for cost reduction. The final output is a set of **data-driven recommendations** for our logistics and operations teams.
## Objectives
- Clean and prepare raw delivery data

- Analyze data using SQL to extract useful insights

- Visualize delivery patterns and metrics in Power BI

- Identify factors impacting delays and delivery success accuracy
## Tools Used
- Excel for data cleaning and formatting
- Sql for data analysis
- Power Bi for data visualization
